# EmotionCoach - 情绪教练

EmotionCoach是一个智能情绪分析和建议系统，它能够：
- 分析用户的情绪状态
- 从空间、时间、任务和人际四个维度进行评估
- 提供个性化的情绪调节建议

## 快速开始

1. 克隆项目
```bash
git clone https://github.com/你的用户名/EmotionCoach.git
```

2. 运行演示版本
```bash
python demo.py
```

## 功能特点

- 多维度情绪分析
- 智能建议生成
- 简单易用的交互界面
- 无需额外依赖

## 使用示例

输入示例：
```
我最近感觉很压抑，工作总是赶不完，人际关系也很紧张
```

系统会分析您的情绪状态，并给出相应的建议。
